Representa una funcion a trozos. Representa una funcion a trozos. Dados una serie de puntos, se crea una funcion a trozos que se puede evaluar para todo numero flotante.

f0 = Y1 si x $<$= X1 f1 = Lineal(Y1, Y2) si X1 $<$ x $<$= X2 f2 = Lineal(Y2, Y3) si X2 $<$ x $<$= X3 ... fn-\/1 = Lineal(Yn-\/1, Yn) si Xn-\/1 $<$ x $<$= Xn fn = Yn si Xn $<$ x

NPoints es el numero de puntos que se va a usar para definir la funcion a trozos
